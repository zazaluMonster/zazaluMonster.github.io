致两年后的自己

发表于 2018-02-22   |   分类于 人生   |     |   阅读次数

致两年后自己的一封信

为了成为永恒,才必须挑战更强大的变化.

两年后的我,你好.

首先,向努力工作的你说声：辛苦了

感觉累吗?有好好休息吗?工作顺利吗?有找到属于自己的那份宝物吗?

现在我刚刚工作没多久,工作稳步上升,觉得一切都进行的稳稳当当.哈,虽然在刚毕业选择了程序员的生活,但是尚不自信自己可否跟上技术进步的脚步(工作忙起来,没什么时间学习新的东西呢).不知道两年后的我是什么状态呢?现在的我还可以对外表示刚工作,所以收入不是很高去解释,但是两年后的我,那时候的我有努力改变好吗?有努力去变的更强吗?

现在的我,对于自己就两个目标,一个是好好工作,还有一个是开发出自己喜欢的一样东西,现在我只考虑是一种网站,类似于gmgard,北+那样,一群喜欢二次元的同僚们一起凑在一起,休息的时候,可以在我的网站里开心畅各个ACG相关的东西.不过我更想去做一份东西,可以拉近我们和二次元人物的距离,让他们更加的生动活泼,使得二次元的世界更加美好

现在刚毕业的我,就已经感觉有好多重要的事情不知道怎么去完成..房子,车子,还有老婆.但是我觉得在这之前,必须要先完成我自己和二次元的那份梦想,因为这样我才有活着的实感,有努力的动力.现在的我,并不觉得房子和车子以及老婆都是必要的东西.甚至这些东西,有些会遮蔽我的感知,让我走到一个奇怪的路上.我希望两年后的我也能保持这份纯真.不被外界影响.

两年后的我如果还是单身,那一定是因为没有遇到同好吧.我知道我自己现在只喜欢和也喜欢二次元的人打交道,不然会感觉少了一份重要的快乐.也许多看点情感篇会好点?现在和女生交流确实太少,感觉已经不太会和女生交流了,哈哈.

哦！对了.现在自己还是比较喜欢听同人音声,两年后的我还会听吗?我被同人音声影响的较多,也在这圈子里知道了好多不错的小姐姐.有部分小姐姐已经成为了我心中憧憬的女友形象(尽管这真的太傻,明知显示基本不可能存在).如果还在听的话,还是少听点R18的哦!多听听ASMR风格的就行了,R18真的洗脑严重呢.

前几天不下心玩300手游冲了648,充完感觉自己特别亏.两年后的我还会怎么盲目吗?两年后的我还会有时间完手游吗?

虽然我不知道两年后的我混的怎么样.但是我希望自己永远不要忘记,自己还有那份未完成的梦想要去实现.

早点睡,周末多学习,少玩游戏呢.

保持睡眠，保持清醒.
